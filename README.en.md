# Tablite

Minimal new tab, focused on search, lightweight and calm. Secure, tracking-free, and ready to use.

Tablite is a Manifest V3 new tab extension for Chromium browsers. It turns the new tab page into a quiet search-first surface: one centered search box, Bing daily image as the background, and configurable search engines. It does not include analytics, ads, or telemetry, keeps settings in local browser storage, and works out of the box with sensible defaults.

> 中文版：[README.md](README.md)

## Features

- Minimal new tab page with no widgets, feeds, cards, or visual clutter.
- Security-conscious by default: built on Manifest V3 with only the permissions needed for local storage and Bing daily image loading.
- No tracking: no analytics, telemetry, ad scripts, or remote configuration.
- Ready to use: install it and immediately get a usable search page, default engines, and a daily background.
- Search engine settings folded into the search box.
- Built-in Google and Baidu search engines.
- Custom search engine URL templates using `%s` as the query placeholder.
- Bing daily image background with local caching and graceful fallback.
- Internationalization through Chrome's official i18n system.
- Built with React, Vite, TypeScript, and Tailwind CSS.

## Install

1. Install dependencies:

```bash
npm install
```

2. Build the extension:

```bash
npm run build
```

3. Load it in a Chromium browser:

- Open `chrome://extensions/`
- Enable Developer mode
- Click Load unpacked
- Select the `dist/` folder in this project

After installation, Tablite replaces the browser's new tab page.

## Security & Privacy

- Minimal permissions: Tablite declares only the `storage` permission and `https://www.bing.com/*` access for loading the Bing daily image.
- Local storage: search engine preferences, custom search URLs, and background cache stay in browser local storage.
- Tracking-free design: Tablite does not collect search history, browsing history, or usage analytics, and does not send data to analytics, ad, or telemetry services.
- Clear network behavior: besides fetching the Bing daily image and opening your chosen search engine when you submit a search, Tablite does not connect to additional services.

## Development


```text
http://127.0.0.1:5173/
```

Common commands:

```bash
npm run dev       # Start the Vite dev server
npm run build     # Build the dist extension bundle
npm test          # Run tests
npm run preview   # Preview the production build
```

Node.js `20.19+`, `22.13+`, or `24+` is required.

## Localization

The single source of truth for copy lives in:

```text
public/_locales/en/messages.json
public/_locales/zh_CN/messages.json
```

Development and test environments generate a fallback file before running:

```text
src/shared/i18n.generated.ts
```

This file is generated by `scripts/generate-i18n.mjs` and is ignored by Git. To update copy, edit only the `messages.json` files under `_locales`.
